# 简介

NoSQL全名Not Only SQL，是非关系型的数据库。是随着访问量的上升，网站的数据库性能出现问题，NoSQL被设计出来。

## 优点

* 高可扩展性
* 分布式计算
* 低成本
* 架构的灵活性，半结构化数据
* 没有复杂的关系

## 缺点

* 没有标准化
* 有限的查询功能
* 不直观的程序
