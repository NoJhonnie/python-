## 数据查询

### 基本查询

* find\(\):查询

```
db.集合.find({条件文档})
```

* findOne\(\):查询一个

```
db.集合.findOne({条件文档})
```

* 方法pretty\(\):将结果格式化

```
db.集合.find({条件文档}).pretty()
```

### 比较运算符

* 等于，默认
* 小于$lt
* 小于或等于$lte
* 大于$gt
* 大于或等于$gte
* 不等于$ne

### 逻辑运算符

* 逻辑与，默认
* 逻辑或$or

```
db.sub.find({$or:[{count:{$gte:10}}],title:"vb"})
```
