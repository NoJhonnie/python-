### ubuntu安装

```
sudo apt-get install mongodb
```

### window安装

详见onenote

### 管理mongo

* 启动

```
sudo service mongod start
```

* 停止

```
sudo service mongod stop
```

* 终端连接

```
mongo
```

* 命令

```
db查看当前数据名称
db.start()查看当前数据信息
```

* 终端退出连接

```
exit
ctrl+c
```

GUI：robomongo，解压后在bin目录下找到运行程序
