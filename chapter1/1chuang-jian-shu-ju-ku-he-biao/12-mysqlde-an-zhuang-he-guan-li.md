## 安装

ubuntu的系统：

```
sudo apt-get install mysql-server mysql-client
```

window系统

下载mysql客户端，mysql官网[https://www.mysql.com/](https://www.mysql.com/)

## 管理服务

启动

```
service mysql start
```

停止

```
service mysql stop
```

重启

```
service mysql restart
```
