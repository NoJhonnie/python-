### GET属性

GET是QueryDict类型的对象，它包含了get请求方式的所有参数，与url请求地址中的参数对应，参数也是位于？后面，格式为key1=value1；多个参数，则使用&连接，格式为key1=value1&key2=value2。

```
def getTest2(request):
    a = request.GET['a']
    b = request.GET['b']
    context = {'a':a, 'b':b}
    return render(request,'booktest/getTest2.html',context)
```
